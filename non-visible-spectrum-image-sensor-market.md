<p><ul><li>Linear Image Sensor</li><li>Area Image Sensor</li></ul></p>
<p><p>The non-visible spectrum image sensor market encompasses two main types: linear image sensors and area image sensors. Linear image sensors capture images in a single row, commonly used in applications requiring fast scanning, such as barcode readers and spectroscopic analysis. In contrast, area image sensors capture images across a two-dimensional grid, making them ideal for applications like security cameras and medical imaging. Both types enable advanced imaging capabilities beyond the visible spectrum, enhancing various industries' analysis and monitoring technologies.</p></p>
